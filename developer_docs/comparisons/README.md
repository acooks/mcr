# Comparisons

This directory contains analysis documents that compare the Multicast Relay (MCR) with other relevant networking tools or approaches. These comparisons highlight MCR's unique advantages and design choices.

*   **[MCR vs. socat](./MCR_vs_socat.md):** A detailed comparison of MCR's architecture, performance, and operational model against `socat`, a general-purpose networking utility, to illustrate where MCR provides superior solutions.
